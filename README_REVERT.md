# REVERT 명령어

reset 명령어와는 달리, 이 명령어는 **원격 저장소** 에 잘못된 커밋이 존재할 때 이를 안전하게 되돌리기 위한
명령어입니다.

- git revert <되돌릴 커밋의 해시 값>

git log --oneline ... 등의 명령어를 쳐 커밋 로그를 보면, 커밋별로 고유 해시 값이 존재합니다.
되돌리려고 하는 커밋의 해시 값을 찾아 명령어에 넣으면 그 커밋을 되돌리게 되며 이것을 또다른 커밋 기록으로 남깁니다.

reset과는 달리, **명확한 변경 사항을 남기므로** 사후 대처에 훨씬 안전합니다.
